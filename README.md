# Stardust Creator - 3D 粒子手势交互系统

基于 **Three.js** 和 **MediaPipe Hands** 的实时 3D 粒子可视化交互网页。通过摄像头识别手势来控制星尘粒子的形态、旋转和扩散效果。支持桌面端与移动端。

## 功能特性

### 手势交互
- **手势控制旋转轴**：伸出单根手指绕 X 轴自转，握拳绕 Z 轴翻滚
- **手掌全开扩散**：五指张开时粒子向外星云扩散
- **摄像头距离感应**：手靠近/远离摄像头可拉近/拉远视角

### 粒子形态
- **四种粒子形态**：土星（带星环）、爱心、花朵、烟花
- 形态切换时粒子平滑过渡变换

### 色彩系统
- **彩虹模式（默认）**：GPU Shader 驱动的流动彩虹色，每个粒子拥有独立色相身份，随时间漂移变换，粒子间自然柔边渐变
- **调色板模式**：自由选择 1~5 种颜色组成自定义渐变色环，支持动态增减颜色
- **单色模式**：经典单色选择

### 其他控制
- **粒子大小调节**：滑块控制粒子点尺寸
- **自转速度调节**：独立控制旋转快慢
- **彩虹流速调节**：控制颜色流动变化速度
- **全屏模式**：支持桌面端与 iOS Safari（webkit 前缀兼容）

### 移动端适配
- 自动检测移动设备，强制使用前置摄像头
- 移动端粒子数自动降至 8000 上限，星空粒子缩减至 800
- 触摸目标适配（按钮最小 44px）
- 刘海屏安全区适配（`safe-area-inset-bottom`）
- 触摸事件拦截，防止页面意外滚动/缩放

## 技术栈

| 技术 | 用途 |
|------|------|
| [Three.js](https://threejs.org/) (r128) | 3D 渲染引擎，自定义 ShaderMaterial 实现 GPU 粒子着色 |
| [MediaPipe Hands](https://mediapipe.dev/) | 实时手势识别与骨骼追踪 |
| GLSL (WebGL 1.0) | 自定义 Vertex/Fragment Shader，HSL 色相漂移与调色板插值 |
| HTML5 Canvas / WebGL | GPU 加速粒子渲染 |
| MediaPipe Camera Utils | 摄像头捕获与管理 |

## 使用方法

1. 用现代浏览器（Chrome/Edge/Firefox/Safari）打开 `index2.html`
2. 点击页面中央的 **"启动"** 按钮
3. 浏览器会请求摄像头权限，点击允许（移动端自动使用前置摄像头）
4. 将单手伸到摄像头前，尝试以下手势：
   - **☝️ 伸出一根手指** → 粒子绕 X 轴旋转
   - **✊ 握拳** → 粒子绕 Z 轴翻滚
   - **🖐️ 五指张开** → 粒子向外扩散
   - **手靠近/远离** → 视角拉近/拉远
5. 使用右侧面板切换形态、色彩模式、大小和速度

## 性能说明

- 粒子数量根据设备自动调整：
  - 移动端：6000 ~ 8000 个
  - 桌面端：6000 ~ 15000 个（按 CPU 核心数）
- 颜色计算全部在 GPU Shader 中完成，零 CPU 开销
- 使用 delta time 动画系统，高刷新率（120Hz+）屏幕不会加速
- 移动端摄像头分辨率自动降至 320x240 以降低 CPU 负载
